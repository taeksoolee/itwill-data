마우스 이벤트

click : 마우스 버튼을 누른 경우 발생되는 이벤트

dblclick : 마우스 버튼을 연속적으로 두번 누른 경우 발생되는 이벤트

mouseenter : 마우스 커서가 엘리먼트 영역에 위치한 경우 발생되는 이벤트

mouseleave : 마우스 커서가 엘리먼트 영역에 위치하지 않은 경우 발생되는 이벤트

mouseover : 마우스 커서가 엘리먼트 영역에 진입한 경우 발생되는 이벤트

mouseout : 마우스 커서가 엘리먼트 영역을 벗어난 경우 발생되는 이벤트

hover : 마우스 커서가 엘리먼트 영역에 진입하거나 벗어난 경우 발생되는 이벤트
