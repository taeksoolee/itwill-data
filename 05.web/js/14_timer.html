내장함수 - timer 함수

setTimeout(callback, ms) : 시간(ms) 경과 후 콜백 함수의 명령을 한번만 실행하는 함수 - timeoutId(함수 호출 고유값) 반환

clearTimeout(timeoutId) : timeoutId로 등록된 setTimeout() 함수를 초기화 시키는 함수 - setTimeout() 함수 실행 취소

setInterval(callback, ms) : 시간(ms) 경과 후 콜백 함수의 명령을 반복 실행하는 함수 - intervalId(함수 호출 고유값) 반환

clearInterval(intervalId) : intervalId로 등록된 setInterval() 함수를 초기화 시키는 함수 - setInterval() 함수 실행 취소
